# transitkit-data

Public CDN data for [TransitKit](https://github.com/andreatoffanello/transit-engine) — a white-label iOS public transit framework.

## Structure

```
appalcart/
  config.json      # Operator configuration (branding, API endpoints, features)
  schedules.json   # GTFS-derived schedule data
```

## Usage

Files are served via GitHub Pages at:
`https://andreatoffanello.github.io/transitkit-data/<operator>/config.json`
